CSS3学习笔记一

Yansuuu 5月 23, 2018
  • 笔记

A.选择器

1.属性选择器

  • E[attr] 含有该属性的E元素
  • E[attr=val] 含有该属性，且属性值完全等于val的E元素
  • E[attr*=val] 属性值包含val就可以了
  • E[attr^=val] 属性值开头含有val就可以了
  • E[attr$=val] 属性值结尾有val就可以了

2.伪类选择器

伪类选择器的标志是 ：

  • E:first-child 找到E元素的父元素中的第一个子元素，要求第一个子元素必须是E元素，否则谁都选不了
  • E:flast-child 同上，父元素中最后一个子元素
  • E:nth-child(n) 同上，n为0,1,2,3,4……，n<=0就无效，然后可以根据2n/2n-1/7n等各种组合来获取目标值，-n+5指的是前五个E元素 还可以填写odd/even来规定奇数、偶数项的样式
  • E:nth-last-child(n) 同上，只不过是从后往前，-n+5是倒数5个
  • E:first-of-type 也是找到E元素的父元素，让后根据E的类型查找序列，也就是说就是第一个E元素，与父元素第一个子元素是谁无关
  • E:last-of-type 同上，最后一个子元素E
  • E:nth-of-type(n)
  • E:nth-last-of-type(n)
  • E:empty 选择完全不含有子元素的E元素
  • E:target 这个选择器要配合锚标签，锚点被激活时样式更换成这个伪类样式

3.伪元素选择器

伪元素标志是 ::

所谓的伪元素就是和html元素功能完全一致，但是不会在DOM树上生成节点

并且所有伪元素都是行内元素，经常要配合display/position/float来设置宽高

  • ::before和::after在css2中是伪类，在css3中是伪元素

    1. 这两个伪类必须要写content属性，即使无需添加内容也需要写content:””
      1. 设置宽高需要转换为块级元素或者行内块
  • ::first-letter 第一个字母或文字，如果需要首字母下沉，需要浮动
  • ::first-line 第一行，会自适应父盒子宽高的变化
  • ::selection 选中的文本，可设置color和background，但是无法这是内容大小

B. 新增颜色模式

1.透明度问题

  • opacity：可以改变透明程度，但是会影响其他属性和子元素
  • transparent ：就是透明，无法更改透明程度，不会影响其他属性和子元素
  • rgba/hsla：能改变透明程度，也不会影响其他属性和子元素

2.rgba()/hsla()

a.rgba()

  • r - red 0-255
  • g green 0-255
  • b blue 0-255
  • a alpha 0-1

rbg也可以设置百分数，但是兼容性不好

b.hsla()

  • h hue色度 0-360 0/360-red 120green 240-blue
  • s saturation饱和度 0.0%-100.0%
  • l lightness 0.0%-100.0%
  • a alpha 0-1

c.text-shadow文本阴影

text-shadow: offsetX offsetY blur color

分别是水平位移，垂直位移 模糊程度 和阴影颜色

下右为正，上左为负；blur没有负数

可设置多个阴影，用,隔开

p {
    text-shadow: -3px -3px 3px #fff, 3px 3px 3px #000;
}

D.盒模型 box-sizing

1.box-sizing: content-box;

外增模式：盒子会因为border和padding撑大

盒子真实大小 = border + padding + width

2.box-sizing: border-box;

内减模式：盒子不会因为border和padding撑大，而是向内压缩内容区大小，也就说width和height不会改变

width/height = border + padding + 内容区宽高

E.浏览器私有前缀（为了兼容性）

  • -webkit- :chrome safari
  • -moz- : firefox
  • -ms-: ie
  • -o- : opera
  • 还要写一个什么都不加的
div {
    background: -webkit-linear-gradient(left, red, blue);
    background: -moz-linear-gradient(left, red, blue);
    background: -ms-linear-gradient(left, red, blue);
    background: -o-linear-gradient(left, red, blue);
    background: linear-gradient(left, red, blue);
}
